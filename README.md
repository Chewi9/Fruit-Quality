# Fruit Quality
Proyecto Visión sobre análisis de frutas

Analizador de las frutas.

Su función será determinar el estado de una fruta. Se le pasará una imagen de esta, y comprobará si presenta alguna deformación. Si esto ocurre, significará que está en mal estado, por lo que marcará la zona donde está mal la fruta y lo notificará para su eliminación. Si no detecta ninguna tara en la fruta, indicará que está en buenas condiciones para su consumo o venta.
